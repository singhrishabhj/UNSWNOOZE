* When the alarm triggers, set the audio volume to **0.5 (50%)**.
* Ask the user for permission to play audio when the app is first used.
* If permission is granted, automatically set alarm audio to 50%.
* Use the HTML audio element with looping enabled.

Example behavior:

When alarm activates:

* audio.volume = 0.5
* audio.loop = true
* audio.play()

FACE VERIFICATION WITH LIVELINESS DETECTION

Improve the face verification wake-up task.

The alarm must only stop when a **real human face is detected and verified as live**.

Implementation requirements:

1. Use the device camera to capture live video.
2. Detect a human face in the frame.
3. Perform basic liveliness checks such as:

* Eye blink detection
* Head movement
* Smile detection

Only after these conditions are met should the alarm be allowed to turn off.

Example verification flow:

* Camera opens
* Face detected
* User blinks or moves head
* Liveliness confirmed
* Alarm stops

If no face is detected or the face is not live, show message:

"Face verification failed. Please try again."
